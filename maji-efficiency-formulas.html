Efficiency Formulas Quick Reference

Core majaco methodologies for measuring manufacturing efficiency, labour productivity, and lost time analysis. All formulas use UK English terminology and time-based measurement principles.

This page was created by maji, majaco's AI operational excellence tool. maji is in active development, so occasional inaccuracies may appear as the system continues to learn.

Core Time Definitions

TermAbbreviationDefinition
Calendar TimeCTTotal time in period (24 × 7 × weeks). Fixed.
Non-working TimeNWTWeekends, bank holidays, planned shutdowns not counted as shift time
Shift Time (Available Time)STCT − NWT. Rostered hours.
Schedule LossScLShifts rostered but not run due to demand
Planned Production TimePPTST − ScL. Time the line was expected to produce.
Planned DowntimePDTScheduled maintenance, changeovers, planned cleaning
Operating TimeOTPPT − PDT. Time the line should have been running.
Unplanned DowntimeUPDTBreakdowns, stoppages, material/labour waiting
Up TimeUTOT − UPDT. Time the line was actually running.
Lost Time Slow RunningLTSRTime equivalent of running below target speed
Lost Time Making WasteLTMWTime equivalent of producing off-spec product
Potential TimePTUT − LTSR − LTMW. Effective productive time.

Lost Time Category Formulas

LTSR = (Target Rate − Actual Rate) / Target Rate × Up Time
LTMW = (1 − Yield) × Potential Time (if waste is made at potential rate)
Total Losses = UPDT + LTSR + LTMW + PDT + ScL
Verification: PPT = PT + UPDT + LTSR + LTMW + PDT

Efficiency Metrics

MetricFormulaDenominator includes
Operating Efficiency PT / OT UPDT + LTSR + LTMW only
Machine Efficiency (OOE) PT / ST All losses including PDT and ScL
OEE (standard) Availability × Performance × Quality UPDT + LTSR + LTMW (same as Operating Efficiency)
TEEP PT / CT All time including non-working time
OEE Availability = UT / OT
OEE Performance = (Actual Output × Ideal Cycle Time) / UT
OEE Quality = Good Output / Total Output
OEE = Availability × Performance × Quality

Labour Efficiency Framework

Used in sales-constrained environments where labour cost drives profitability:

PPLH = (Good Output) / (Total Labour Hours Worked)
PPLH = Machine Efficiency × (Potential Crew / Actual Crew) × Target Rate
Labour Efficiency = PPLH Actual / PPLH Target

Where Potential Crew is the minimum number of people needed to run the line at target rate.

Financial Valuation of Lost Time

Constraint StateValue of lost potential time
Production-constrained (demand > capacity) Lost PT × Throughput Rate (revenue − truly variable costs)
Sales-constrained (capacity > demand) Lost PT × Labour Cost Rate (savings from fewer people/hours)
Throughput = Revenue − Truly Variable Costs (materials, packaging, energy)
Value of 1 hour constrained downtime = Throughput Rate × 1 hour
